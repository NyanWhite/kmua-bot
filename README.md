## 部署

### 源码运行

1. `git clone https://github.com/krau/kmua-bot`
2. pip install -r requirements.txt
3. edit config.yml
4. python3 bot.py

### 使用docker compose

1. `git clone https://github.com/krau/kmua-bot`
2. edit config.yml
3. `docker compose up -d`
