## 部署

### 源码运行

1. `git clone https://github.com/krau/kmua-bot`
2. pip install -r requirements.txt
3. edit settings.toml
4. python bot.py

### 使用docker compose

1. `edit docker-compose.yml`
2. `docker compose up -d`
